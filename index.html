Rasim.jl

Rasim (RAdio network SIMulator) is a simple time-slot-based simulator for cognitive radio networks. Rasim.jl is a port of rasim to Julia. I created the port mainly because the performance (simulation time) I was getting in Python was not satisfactory for my graduation project. As I added new features for my proposed algorithm, Rasim.jl became more complicated than and a successor of Rasim.

You can change simulation parameters in Params.jl and add your own agents in seperate Julia files. To simulate your new agent type, add it to the agent_types list in Rasim.jl. Then, you can run the simulation by running

julia Rasim.jl

in the command line.
